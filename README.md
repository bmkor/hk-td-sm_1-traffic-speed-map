# hk-td-sm_1-traffic-speed-map
Handy downloader written in `golang` for historical speed data in https://data.gov.hk/en-data/dataset/hk-td-sm_1-traffic-speed-map

# Dependency
* `go get github.com/bmkor/gopushbullet`
